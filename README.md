#  Event Navigator – College Event Management System

Event Navigator is a modern, user-friendly event management website designed specifically for college students.  
It helps students *discover, register, track, and manage college events* from a single platform.

The application follows a *pink–purple–blue–orange gradient theme* and focuses on a *clean, stylish, responsive, and engaging UI* for an excellent user experience across all devices.

---

##  Project Overview

Event Navigator simplifies the entire college event ecosystem by:
- Centralizing all events, recruitments, and announcements
- Helping students avoid event clashes
- Providing personalized event suggestions
- Improving communication between students and societies

---

##  Authentication & User Registration

### Sign In / Sign Up Features
- Secure authentication system
- Allowed email format:
- Password requirements:
- Minimum 8 characters
- At least one uppercase letter
- At least one lowercase letter
- At least one number
- At least one special character

### User Details Collected
- Username
- Programme (e.g., B.Tech)
- Branch (e.g., CSE)
- Year (1st to 4th year for B.Tech)
- Passing year (e.g., 2027)
- Mobile number (optional)

### Verification
- Email verification
- Mobile number verification

##  Home Screen Features

1. *Search Bar*
 - Search events using filters such as category, popularity, and date

2. *Notifications & Alerts*
 - Notifications for:
   - New events
   - New notices
   - Updates in registered events

3. *Event Calendar*
 - Green mark: dates with unregistered events
 - Red mark: dates with at least one registered event

4. *Dashboard*
 - User profile overview
 - Registered event details
 - Event tracking with date and time
 - Detection of clashes between two events

5. *Event Suggestions*
 - Personalized event recommendations based on:
   - Previously attended events
   - User interests and positive feedback

6. *Popular Events Section*
 - Displays trending and recent events
 - Based on number of votes received by societies

7. *Feedback Pop-ups*
 - Feedback requests for events the user has participated in
